# 💚 Relationship Research Dashboard

A cinematic, data-driven Streamlit app designed as a **“scientific analysis report” of a relationship**.

This project blends:
- data visualization
- storytelling
- interactive UI
- maps + memories
- playful pseudo-science
- personal photos and video

The result is a **birthday experience disguised as an environmental / emotional research system**.

---

# 🎯 Concept

The app simulates a fictional research institute analyzing a relationship as if it were an ecological / emotional dataset.

Everything is presented as:
> “Longitudinal Emotional Impact Study”

But in reality, it is a personalized interactive love letter.

---

# 🧠 Core Idea Modules

## 1. 🎬 Intro Video (Research Director Message)
- A short personal video of you
- Introduces the “research report”
- Sets emotional tone
- Acts as cinematic entry point

**Goal:** transition user from reality → immersive experience

---

## 2. 📊 Relationship KPI Dashboard
A fake scientific dashboard showing “metrics” like:
- Relationship Stability
- Emotional Dependence Index
- Shared Adventures Count
- Happiness Levels
- Probability of Missing Each Other

**Goal:** humor + data aesthetic

---

## 3. ⚠️ Risk Assessment (Ecotoxicology Style)
Styled like environmental safety analysis:
- Emotional exposure levels
- Cognitive stability (joke metric)
- Attachment persistence
- “Safe / Elevated / Critical” states

**Goal:** playful scientific exaggeration

---

## 4. 🧾 Timeline of Memories
Interactive chronological story:
- first meeting
- trips
- dates
- random small moments

Each entry includes:
- photo
- description
- optional location
- emotional tag

**Goal:** emotional core of the project

---

## 5. 🔮 Pseudo Future Extrapolation
Fake predictive analytics:
- relationship “trend line”
- happiness forecast
- future milestones (joke-based)

**Goal:** humor + visual storytelling via data

---

## 6. 🧪 Field Observations
Scientific journal-style entries describing behavior:
- funny daily moments
- habits
- “observed phenomena”

Format:
> Observation #12: Subject shows increased happiness after coffee exposure.

**Goal:** playful documentation style

---

## 7. 🗺️ GIS Memory Map
Interactive map with:
- places you visited together
- clickable markers
- popups with stories and images

Each location includes:
- what happened there
- photos
- short “field note”

**Goal:** spatial memory storytelling

---

## 8. 🖼️ Photo Gallery
Curated collection of shared images:
- polaroid style or grid layout
- grouped by events or moods
- optional captions

**Goal:** visual nostalgia

---

## 9. 📄 Fake Research Paper (PDF Export)
Downloadable scientific-style report:
- abstract
- findings
- conclusions
- humorous scientific tone

Generated as PDF using ReportLab.

**Goal:** final “official documentation”

---
